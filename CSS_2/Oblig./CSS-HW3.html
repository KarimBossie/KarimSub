Блок навигации

  • Ширина 15em
  • Должен присутствоать цвет фона
  • Располагается слева
  • Блок навигации наполните контеном достаточным, чтобы по высоте он начинал захватывать вторую статью, как показано в данном примере
Статья 1
  • Занимает всю доступную ширину
  • Должен присутствоать цвет фона
  • Во всех "статьях" сделайте отступ снизу от соседей равным .2ем, при помощи свойства margin-bottom.
Статья 2
  • Размер блока:
  • Занимает всю доступную ширину
  • Должен присутствовать цвет фона
  • Во всех "статьях" сделайте отступ снизу от соседей равным .2ем, при помощи свойства margin-bottom. Картинка вставлена при помоще тега, плавание организовано по правому краю внутри данной статьи для этой картинки

Нижний блок, подвал. Занимает всю доступную ширину. Высота определяется помещенным внутрь контентом.
